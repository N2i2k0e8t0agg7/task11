# Compound Interest Calculator

## Objective
This project is a simple JavaScript program that calculates compound interest using predefined variables.

## Formula Used
A = P * (1 + r / n) ^ (n * t)

Where:
- A = Final Amount
- P = Principal Amount
- r = Annual Interest Rate
- n = Number of times interest is compounded per year
- t = Time in years

## Files Included
### 1. index.html
Loads the JavaScript file into the browser.
### 2. script.js
Contains the compound interest calculation logic.
### 3. style.css
simple css for more presentable
